================================================================================
                          PWA (PROGRESSIVE WEB APP)
================================================================================

What's a PWA?
  A web app that works like a native app. Installable, offline-capable,
  appears on home screen. Not an app store required.

Install on iPhone:
  1. Safari → Share button
  2. "Add to Home Screen"
  3. Creates app icon on home screen
  4. Opens full-screen like a native app

Install on Android:
  1. Chrome menu → "Install app"
  2. Creates app icon on home screen
  3. Opens full-screen like a native app

Offline Support:
  Once you visit the app, it caches the assets. You can then:
  - Load the page offline
  - See cached data
  - Refresh to update when back online

================================================================================
                          TROUBLESHOOTING
================================================================================

NO LIVE DATA?
  Problem: Settings shows API key but data still loading
  Solution:
    1. Verify API key is correct (test at finnhub.io)
    2. Refresh page
    3. Check browser console (F12) for errors
    4. Ensure you're not rate-limited

DETAIL VIEW NOT OPENING?
  Problem: Tap ticker, nothing happens
  Solution:
    1. Try in incognito/private mode
    2. Clear browser cache
    3. Check console for JavaScript errors

BRIEFING.JSON NOT UPDATING?
  Problem: GitHub Actions workflow failed
  Solution:
    1. Check Actions tab in GitHub repo
    2. Verify FINNHUB_API_KEY secret is set
    3. Check if market is open (workflow only runs weekdays)
    4. Verify API key is valid

SERVICE WORKER NOT CACHING?
  Problem: Page doesn't load offline
  Solution:
    1. Clear site data (browser settings)
    2. Uninstall and reinstall PWA
    3. Refresh page while online
    4. Service worker should now cache

ICONS NOT SHOWING?
  Problem: App icon appears blank
  Solution:
    1. Ensure assets/icon-192.png and icon-512.png exist
    2. Wait 24 hours for cache to clear
    3. Try different device
