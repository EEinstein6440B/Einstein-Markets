# Einstein Markets v2.0 - Quick Start Guide

## 60-Second Setup

1. **Get an API Key** (2 minutes)
   - Go to https://finnhub.io
   - Sign up for free (email only)
   - Copy your API key from the dashboard

2. **Run the App** (1 minute)
   ```bash
   npx http-server
   ```
   Open `http://localhost:8080`

3. **Add Your API Key** (30 seconds)
   - Tap the **Settings** tab
   - Paste your Finnhub API key
   - Tap "Save API Key"
   - Wait 5 seconds for data to load

Done! You now have live market data.

## What You Can Do

### Tab 1: Briefing (Main)
- **Market Pulse Strip** at top - tap any ticker for details
- **Market Mood Banner** - current market sentiment
- **What to Watch** - AI-generated alerts
- **Briefing Sections** - organized by theme:
  - AI Infrastructure (10 tickers)
  - Energy & Commodities + Crypto (4 tickers)
- **Big Movers** - highlighted red/green if moved 3%+

### Tab 2: Tickers
- Full grid view of all 14 watchlist tickers
- Tap any ticker for detailed view

### Tab 3: Settings
- Manage your Finnhub API key
- See app version (v2.0)

## The Detail View

Tap any ticker to slide in a detail panel with:

1. **Price Hero**
   - Symbol and price
   - 24h change in green or red

2. **Market Data Grid**
   - Open, High, Low, Prev Close

3. **Investment Thesis**
   - Why we're watching this ticker
   - Key catalysts and risks

4. **AI Analysis Box**
   - Signal: Strong Buy / Buy / Neutral / Caution / Bearish
   - Commentary explaining the signal
   - Analyst consensus bar (colored segments)
   - Analysis timestamp

5. **Latest News**
   - Top 5 company news items
   - Source and date

## Mobile & PWA

This is a Progressive Web App (PWA):

### Install on Home Screen

**iPhone:**
1. Safari → Share → Add to Home Screen
2. Creates app icon on home screen

**Android:**
1. Chrome menu → Install app
2. Creates app icon on home screen

### Offline Support
- Once you visit, the app caches
- Works partially offline (uses cached data)
- Refreshes when you're back online

## Data Refresh

- **Live quotes** - Update every 5 minutes from Finnhub
- **Briefing.json** - Updates twice daily at 6 AM and 4 PM ET
- **News** - Updates when you refresh the page

## API Rate Limits

Finnhub free tier: **60 requests per minute**

The app uses:
- ~14 requests for stock quotes
- ~14 requests for analyst data (if available)
- ~14 requests for company news

This is well within limits. No charges ever.

## Troubleshooting

### No live data?
1. Go to Settings tab
2. Paste your Finnhub API key
3. Wait 5 seconds
4. Refresh the page

### Detail view not working?
1. Try closing and reopening
2. Clear browser cache
3. Refresh the page

### PWA install not showing?
1. Must be HTTPS (not localhost)
2. Must have valid manifest.json (included)
3. Works best on mobile devices

### Briefing.json doesn't update?
- This is auto-generated by GitHub Actions
- Only works if deployed to GitHub with workflow setup
- For local testing, use placeholder data
